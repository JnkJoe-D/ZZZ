# 指令中枢测试流程

## 测试前确认

- 使用角色资源：`安比`
- 角色配置资源已挂接：
  - `安比.asset`
  - `安比_指令上下文.asset`
- 调试 HUD 已打开，观察以下字段：
  - `Command Context`
  - `Last Route`
  - `Route Detail`
  - `EXECUTION HISTORY`

## 1. 待机普攻起手

### 操作

- 角色静止不动
- 点一次普攻

### 期望

- 角色从 `GroundIdle` 进入 `CharacterSkillState`
- 播放 `安比_普通攻击1`
- HUD 中：
  - `Command Context` 从 `GroundIdle` 切到 `Skill`
  - `Last Route` 显示 `ContextRoute / BasicAttack/Started`
  - `Route Detail` 显示 `GroundIdle / - / 10001`

## 2. 普攻连段

### 操作

- 在普攻 1、2、3、4 的有效窗口内继续点普攻

### 期望

- 连段顺序为 `1 -> 2 -> 3 -> 4`
- HUD 中每次连段应显示：
  - `Last Route` 为 `LocalRoute / BasicAttack/Started`
- `普通攻击3` 长按派生时：
  - `Last Route` 为 `LocalRoute / BasicAttack/Performed`

## 3. 地面特殊技起手

### 操作

- 待机或慢跑时按特殊技

### 期望

- 从 `GroundIdle` 或 `GroundJog` 进入 `CharacterSkillState`
- 播放 `安比_特殊技`
- HUD 中：
  - `Last Route` 为 `ContextRoute / SpecialAttack/Started`

## 4. 地面闪避分流

### 操作

- 静止时按闪避
- 有方向输入时按闪避

### 期望

- 无方向输入时优先播放后闪
- 有方向输入时优先播放前闪
- HUD 中：
  - `Last Route` 为 `ContextRoute / Evade/Started`
- 说明：
  - 输入层虽然仍有 `EvadeFront/EvadeBack` 两个回调，但进入中枢后已统一为 `Evade`

## 5. 前闪后接冲刺

### 操作

- 推住移动方向并按闪避
- 保持方向到前闪进入后摇

### 期望

- 先进入 `CharacterEvadeState`
- 在进入 `Backswing` 的那个时刻已经有方向输入时，前闪结束后回到地面并进入 `GroundDash`
- Dash 播放由状态动作指令驱动，HUD 中可看到最近一次 `StateAction`

## 6. 冲刺中普攻

### 操作

- 进入 `GroundDash` 后按普攻

### 期望

- 当前安比配置会从 `GroundDash` 通过 `ContextRoute` 进入 `安比_冲刺普攻`
- HUD 中：
  - `Last Route` 为 `ContextRoute / BasicAttack/Started`
  - `Route Detail` 显示 `GroundDash / - / 10013`

### 说明

- 当前这版测试数据为了先跑通整条中枢链，`DashAttack` 走的是 `GroundDash` 的 `ContextRoute`
- 架构本身已经支持把它改成 `Dash` 动作自己的 `ComboWindow + LocalRoutes`
- 如果后面给 `安比_冲刺` 的时间轴补了 `ComboWindow`，并在该动作资产上补 `LocalRoutes`，那么这里预期会变成：
  - `Last Route = LocalRoute / BasicAttack/Started`

## 7. 后摇取消

### 操作

- 普攻、特殊技或闪避进入 `Backswing` 后
- 在后摇内按普攻、特殊技、闪避

### 期望

- 输入应按 `Backswing` 的 `ContextRoute` 处理，而不是被当前还在播放的旧动作挡住
- HUD 中：
  - `Last Route` 为 `ContextRoute`
  - `Route Detail` 显示 `Backswing`

### 追加检查

- 在 `Backswing` 内只推移动方向、不按攻击键
- 期望立即回到地面 locomotion
  - 有方向时进入 `GroundJog`
  - 无方向时回 `GroundIdle`

## 8. 晚按移动不应接 Dash

### 操作

- 前闪时不要推方向
- 等进入 `Backswing` 以后再开始推方向

### 期望

- 不应进入 `GroundDash`
- 回到地面后应按普通地面逻辑进入 `GroundJog` 或 `GroundIdle`
- 说明：
  - Dash 资格现在是在进入 `Backswing` 那一刻一次性锁定
  - 后摇里晚到的移动输入不会再补拿到 Dash 资格

## 失败时优先看什么

- `Last Route` 没变化：
  - 说明命令没被中枢消费，先看 `COMMAND BUFFER` 里是否进了对应 `CommandType/Phase`
- `COMMAND BUFFER` 有命令但不出招：
  - 看当前 `Command Context` 是否和预期一致
  - 看当前动作是否已经拥有该命令的 `LocalRoutes`
- 出招错了：
  - 对照 `Route Detail` 看命中的是 `LocalRoute` 还是 `ContextRoute`

## 当前结论标准

如果下面 6 条都符合预期，可以认为这版指令中枢架构已经跑通：

1. `Idle -> 普攻1` 走 `ContextRoute`
2. `普攻连段` 走 `LocalRoute`
3. `Idle/Jog -> 特殊技` 走 `ContextRoute`
4. `Evade` 能根据方向分流前后闪
5. `FrontEvade -> GroundDash` 能自动续接
6. `GroundDash -> 冲刺普攻` 能通过中枢进入正确动作

额外回归点：

1. `Backswing` 内按普攻/特殊技/闪避，应走 `Backswing ContextRoute`
2. 前闪进入 `Backswing` 后才补推方向，不应再进 `GroundDash`
